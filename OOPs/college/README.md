This package demonstrates **Java access modifiers within the same package**.

## Classes

### `Student`
Demonstrates all four access modifiers:
- `private`
- default (package-private)
- `protected`
- `public`

Includes variables and methods for each modifier to show visibility rules.

### `Test`
Demonstrates **access within the same package** using the `Student` class.

## Key Observations
- `private` members are accessible only within the same class
- default, protected, and public members are accessible within the same package

## Purpose
This package is designed for:
- Understanding access modifiers
- Exam preparation
- Interview concept clarity
